FireFlight App

This is a web and iOS app that allows users to detect fires near their location, track the fire locations, and get alerts when a fire is within a certain radius of them. This is built using React and Node, as well as utilizing a Flask backend built by our Data Scientists. Mapbox API was used to build out the map and Hooks + Context were used for state management.
